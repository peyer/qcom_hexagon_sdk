Clips 16-bit signed values from input array and store results to the same-size output array. If particular member is greather then high_level parameter, its value is set to high_level. If member's value is lower then low_level parameter, value of that member is set to low_level.

Parameters
[in]samplesInput array of 16-bit values for clipping. Array has to be aligned to 8 bytes.
[out]clippedOutput array of clipped 16-bit values. Array has to be aligned to 8 bytes.
[in]sizeNumber of elements of input and output arrays.
[in]low_levelMinimum clipping value.
[in]high_levelMaximum clipping value.
Returns
Function status: 0 on successful execution, -1 on invalid input data.
