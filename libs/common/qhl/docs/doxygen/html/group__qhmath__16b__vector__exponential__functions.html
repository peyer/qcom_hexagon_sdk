int32_t qhmath_exp10_ah ( int16_t *  input,
int16_t *  output,
uint32_t  size 
)

Computes base-10 exponential over the elements of 16-bit input array.

Parameters
[in]input16-bit fixed-point array consisting of power arguments.
[out]output16-bit fixed-point where will be stored result for each element of input array.
[in]sizeNumber of elements of input and output arrays.
Returns
Function status: 0 on successful execution or -1 if some error occurred.
Note
Input arguments are in Q15 format, which means they are in range [-0.9999, 0.9999]. Max value in output array is 20480 in Q4.11 signed format. Min value in output array is 204 in Q4.11 signed format.
