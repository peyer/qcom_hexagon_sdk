Vectorized copysign function for floating-point values. Fuction composes an output floating-point array from input array by modifying sign of each input element to sign of y.

Parameters
[in]inputInput array of floating-point values from which magnitude is taken.
[in]outputOutput array where modified floating-point values will be stored. Array has to be aligned to 8 bytes.
[in]sizeNumber of elements of input and output arrays.
[in]yFloating-point value from which sign is taken.
Returns
Function status: 0 on successful execution, -1 on invalid input data.

Vectorized modexp function for floating-point values. Fuction composes an output floating-point array from input array by modifying exponent of each input element to k.

Parameters
[in]inputInput array of floating-point values from which sign and mantissa are taken. Array has to be aligned to 8 bytes.
[in]outputOutput array where modified floating-point values will be stored. Array has to be aligned to 8 bytes.
[in]sizeNumber of elements of input and output arrays.
[in]kExponent (base 2) which apply to all output members.
Returns
Function status: 0 on successful execution, -1 on invalid input data.
