Computes the value of base raised to the power exponent over the elements of input array and store results to the output array.

Parameters
[in]inputInput array of bases as single-precision floating-point values. Array has to be aligned to 8 bytes.
[in]exponentExponent as single-precision floating-point value.
[out]outputOutput array where computed values will be stored. Array has to be aligned to 8 bytes.
[in]sizeNumber of elements of input and output arrays.
Returns
Function status: 0 on successful execution, -1 on invalid input data.
