Interpolation IIR - floating-point.

Parameters
[in]in_samples- input samples in time domain
[in]coefs- IIR filter coefficients - arranged as b0, b1, a1, b2, a2
[in,out]states- pointer to filter states buffer
[in]num_biquads- number of consecutive IIR filter calls (with different coeffs)
[in]length- length of input data (number of samples); maximum supported is (512 / r)
[in]r- upscalling factor, minimum 1
[out]out_samples- output buffer; with length (input length * r)
Note
  • Assumptions:
    1. states buffer should be aligned to 8 bytes

[in]r- upscalling factor, minimum 1
[out]out_samples- output buffer (Q15 format); with length (input length * r)
