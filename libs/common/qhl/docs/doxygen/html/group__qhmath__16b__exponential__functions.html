int16_t qhmath_exp10_h ( int16_t  x)

Computes the 10 raised to the given argument.

Parameters
[in]x16-bit fixed-point argument.
Returns
Base-10 exponential of x.
Note
Input argument is in Q15 format, which means that it is range [-0.9999, 0.9999]. Max value of output is 20480 in Q4.11 signed format. Min value of output is 204 in Q4.11 signed format.

Computes the 2 raised to the given argument.

Parameters
[in]x16-bit fixed-point argument.
Returns
Base-2 exponential of x.
Note
Input argument is in Q4.11 signed fixed-point format, which means that it is range [-16, 15.9995]. Max value of output is (2^32 - 1) in Q16.16 unsigned format. Min value of output is 0 in Q16.16 unsigned format.
uint32_t qhmath_exp_h ( int16_t  x)

Computes the exponential value to the given 16-bit fixed-point argument.

Parameters
[in]x16-bit fixed-point argument.
Returns
Exponential value of x.
Note
Input argument is in Q4.11 signed fixed-point format. Max value of output is (2^32 - 1) in Q16.16 unsigned format. Min value of output is 0 in Q16.16 unsigned format. If the input value greater then 22712, output is set to max value 0xFFFFFFFF
