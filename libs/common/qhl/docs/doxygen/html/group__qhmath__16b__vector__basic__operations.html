Computes absolute value of each element in 16-bit fixed-point input array and returns array of results.

Parameters
[in]inputInput array of 16-bit fixed-point values. Array has to be aligned to 8 bytes.
[out]outputOutput array where computed absolute values will be stored. Array has to be aligned to 8 bytes.
[in]sizeNumber of elements of input and output arrays.
Returns
Function status: 0 on successful execution, -1 on invalid input data.
