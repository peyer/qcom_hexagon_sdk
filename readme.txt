****************************************************
Hexagon SDK
****************************************************

The Qualcomm Hexagon SDK enables developers to 
leverage the Qualcomm Hexagon DSPs for their 
applications.

You can get started with the SDK tools and examples 
by reading docs/index.html from a browser. 
You may also want to read the DSP User Guide, 
which is located at 
docs/images/80-VB419-108_Hexagon_DSP_User_Guide.pdf. 
This will introduce you to the DSP architecture, 
optimization techniques, and SDK tools available 
with the Qualcomm Hexagon SDK.
