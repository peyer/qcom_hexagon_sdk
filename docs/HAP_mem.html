DSP User PD Memory Management

HAP Mem API's

HAP_mem_set_grow_size

Overview

The HAP Mem APIs allow the user to manage the user PD heap memory and control the mapping of buffers into this memory.

HAP Mem API's

HAP_mem_get_stats

Get user PD heap current statistics.

API

int HAP_mem_get_stats (struct HAP_mem_stats* stats)

Return

Returns 0 for success, non-zero value otherwise.

HAP_mem_stats structure members are of type uint64 and are described in table below.

bytes_free

number of bytes free from all the segments

bytes_used

number of bytes used.

seg_free

number of segments free.

seg_used

number of segments used.

min_grow_bytes

minimum number of bytes to grow the heap when creating a new segment.

HAP_mem_set_grow_size

Set the minimum and maximum grow size. The heap will grow the memory by at least min and at most max bytes.

API

int HAP_mem_set_grow_size (uint64 min, uint64 max)

min

minimum number of bytes by which to grow the heap when requesting a new segment.

max

maximum number of bytes by which to grow the heap when requesting a new segment.

Return

Returns 0 for success, non-zero value otherwise.

HAP_mmap

Map buffer associated with the file descriptor to memory. The reference count gets incremented if the file descriptor is already mapped.

API

void* HAP_mmap (void* addr, int len, int prot, int flags, int fd, long offset)

addr

non-NULL address to request mapping to this location, NULL otherwise.

len

size of the buffer to be mapped.

flags

HAP_MAP_NO_MAP to increment reference count with no mapping, 0 to map the buffer and increment the reference count.

fd

file descriptor for the buffer.

offset

offset into the buffer.

Return

-1 on failure, 0 on success and HAP_MAP_NO_MAP flag set, mapped address otherwise.

HAP_munmap

Decrease the reference counter. Unmap the buffer from memory if the reference count reached 0.

API

int HAP_munmap (void* addr, int len)

addr

mapping address.

len

size of the mapped buffer.

Return

Returns 0 for success, non-zero value otherwise.

HAP_mmap_get

Get virtual and physical address associated with the buffer and increment the reference count.

API

int HAP_mmap_get (int fd, void** vaddr, void* paddr)

fd

file descriptor for the buffer.

vaddr

virtual address associated with the buffer.

paddr

physical address associated with the buffer.

Return

Returns 0 for success, non-zero value otherwise.

HAP_mmap_put

Decrements the reference count of the file descriptor.

API

fd

file descriptor for the buffer.

Return

Returns 0 for success, non-zero value otherwise.
