 This is a Thread Local Storage example.  In multithreaded architectures, it is
 advantageous to have a static local or global variable that is thread
 specific.  This example demonstrates how to initialize a thread local variable.

 The example will create two new hardware thread tasks that point to the same
 function but with thread local variables.

 To clean, build and simulate the example type:  make
