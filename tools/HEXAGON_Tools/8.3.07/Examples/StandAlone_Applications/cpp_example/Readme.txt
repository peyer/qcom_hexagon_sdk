 This is a c++11 example that demonstrates the use of a range iterator.  The
 objective of this example is to show that c++11 extensions are supported and
 also to demonstrate how to build and simulate the example.

 All c++ examples should be build/simulated with hexagon tools version 8.0 or
 higher to work correctly.

 To build and simulate the example type:  make
